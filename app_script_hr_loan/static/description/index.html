Overview

By using this module, you can actively and transparently monitor all operations in loan process,, This module provides employee loan management functionality. You can configure many loan types, restrict employee loan request, multiple employee loan, payroll integrated and many more features. Manage your employee loans with our Loan Management module. View all the current running loans, approved and rejected employee loan requests.

Configuration

  • Configure Your loan types.

Features

  • Any employee can request a loan..
  • Set different Loan Types.
  • Ability to calculate loan amount based on employee salary or fixed amount.
  • Ability to calculate with decimal part or without it.
  • Control loan type with max amount allowed, and max number of installments allowed.
  • Set Limit for employee loan request.
  • provide Loans interference for employee, if you want.
  • Connect loan type with violation years of employment as configration.
  • Ability to compute installments, high flexibility to configure number of installment and automatic computation of installment per months.
  • Loan installment can be Suspended based on configration.
  • Ability to restructuring employee loan, you can pay partial amount and divide the remaining on number of Installments.
  • Ability to pay selected installments or all installments of loan directly and also you can reduce installments amount.
  • Specify loan account and journal per type, as configuration.
  • Fully integrated with finance and payroll modules.
